# Simple Multi-Agent RAG Demo

This Python script demonstrates a simplified Retrieval-Augmented Generation (RAG) system in a multi-agent simulation context. It shows how different agents can query a shared knowledge base, retrieve relevant information, and use it to augment their responses, mimicking how LLMs overcome knowledge cutoffs with external data.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
